# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-XX-XX

### Added
- Initial release of GitSage
- Support for OpenAI compatible APIs
- Model list fetching via v1/models endpoint
- Multiple commit conventions (Conventional Commits, Angular, Emoji, Simple)
- Multi-language support (English, Chinese, Auto)
- Integration with Git commit tool window
- Application-level settings with secure API key storage
- IDE proxy support
- Customizable temperature and max tokens
